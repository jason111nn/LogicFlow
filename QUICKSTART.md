# LogicFlow 快速啟動指南

## 安裝步驟

1. **確保已安裝 Node.js**
   ```bash
   node --version  # 應該顯示 v18 或更高版本
   ```

2. **安裝專案依賴**
   ```bash
   npm install
   ```

3. **啟動開發伺服器**
   ```bash
   npm run dev
   ```

4. **在瀏覽器中開啟**
   - 通常會自動開啟 `http://localhost:5173`
   - 如果沒有自動開啟，請手動在瀏覽器中輸入該網址

## 常見問題

### Q: npm install 失敗？
A: 請確保：
- Node.js 版本 >= 18
- 網路連線正常
- 嘗試清除快取：`npm cache clean --force`

### Q: 開發伺服器無法啟動？
A: 檢查：
- 端口 5173 是否被占用
- 是否有其他錯誤訊息
- 嘗試使用 `npm run dev -- --port 3000` 指定其他端口

### Q: 樣式沒有正確顯示？
A: 確保：
- Tailwind CSS 已正確安裝
- 瀏覽器支援 CSS Grid 和 Flexbox
- 清除瀏覽器快取

### Q: 截圖功能無法使用？
A: 檢查：
- html2canvas 是否正確安裝
- 瀏覽器是否允許下載檔案
- 是否有 CORS 限制（本地開發通常沒問題）

## 開發建議

1. **擴充題庫**：編輯 `src/utils/questionGenerator.ts` 添加更多題目
2. **自訂樣式**：修改 `tailwind.config.js` 或組件內的 className
3. **新增章節**：在 `src/types/chapters.ts` 中添加新章節定義
4. **調整功能**：各組件都在 `src/components/` 目錄下
